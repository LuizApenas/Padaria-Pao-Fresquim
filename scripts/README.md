# Scripts

Automação auxiliar para gestão do repositório e do backlog.

## Arquivos atuais

- `criar-issues.py`: criação automatizada das issues da Fase 1 via GitHub CLI
- `criar-issues.sh`: alternativa shell para a mesma rotina

## Pré-requisitos

- GitHub CLI (`gh`)
- autenticação com permissões de `repo` e `project`, quando necessário
